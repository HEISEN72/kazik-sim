# kazik-sim
Тренировочный пет проект, больше в readme
Суть проекта: это сайт-симуляция казино с ненастоящими деньгами. В ней есть механики рандомной рулетки, баланса, выбора ставки, истории выигрышей. Есть система аккаунтов, при заходе на сайт можно зарегистрироваться или войти, нужны ник, тг ID (чтобы бот выслал код с подтверждением) и пароль. Все данные хешируются в базу данных. Напрямую вы можете ознакомиться с проектом по ссылке HEISEN7.pythonanywhere.com (заходить с впном)
Изначально я написал простой код когда начинал изучать пайтон. Недавно я решил доработать его с помощью Gemini Pro 3.1 и вывести на уровень рабочего сайта. Под моим руководством ИИ с этой задачей справился, я находил баги и ошибки, писал ему и он исправлял их. Результатом грамотной работы с ИИ как с инструментом и стал этот проект
